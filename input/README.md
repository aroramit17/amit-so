# input/

Drop your resume here as one of:

- `resume.pdf`
- `resume.docx`
- `resume.txt`
- `resume.md`

Then open this project in [Claude Code](https://claude.com/claude-code) and run:

```
/populate-from-resume
```

Claude will read your resume and populate `src/data/site.js` for you — bio, experience, skills, certifications, projects, and milestones — then walk you through the remaining manual steps (your portrait, your domain, your Cal.com booking link).

Files in this directory (other than this README) are gitignored, so your resume will not be committed.
